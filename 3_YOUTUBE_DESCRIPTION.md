## What you'll build

A personal AI agent that:

- Connects to your real tools (Gmail, Calendar, Slack, Notion, Linear, +1000 more) via Composio
- Remembers things about you across sessions and channels via Supermemory
- Has a customizable personality (a "soul") that the agent helps you write through conversational onboarding
- Replies from Telegram so it's not stuck in a browser tab
- Runs scheduled tasks (cron) so it can act without you opening the app

All of it on top of the Vercel AI SDK chatbot template, deployed on Vercel.

## FAQ (short)

**Do I need to know how to code?** No, not as an expert. You should be able to read code, review what an AI agent writes, and steer it. We use Cursor in the video but the project is agent-agnostic — Claude Code, Codex, Windsurf, OpenCode all work with the same prompts.

**Do I need a paid AI coding tool?** No. Cursor has a free Hobby plan, and students can get a year of Cursor Pro free. Claude Code may need a paid plan; check Claude for Education or Codex for Students if you're a student. Worst case, OpenCode is fully free.

**Does Composio / Supermemory cost money?** Both have free tiers that cover this whole tutorial. Composio is free for individual builders. Supermemory's free plan = 1M tokens + 10K queries/month. If you're an early-stage startup and need more, apply to the Composio Startup Program (link above).
