# Overall Tutorial Structure

### Core Features

- **Chatbot:** First, we’ll clone the chatbot from Vercel.
    - Database, caching, AI models, chat SDK, tool calling out of the box!
    - Deploy the app with 1 click
    - Explain overall chat architecture - but we gloss over most of it.
- **Tools:** Give our chatbot hands with Compsio.
    - Connect to 1000+ tools.
        - Business ⇒ Email + Linear + Slack
        - Fun ⇒ Instagram
        - Docs ⇒ Notion
    - Preconfigure → then show them the wow moment!
- **Memory:** We wrap everything in Supermemory so it can remember stuff about us.
    - This is really simple; keeps it light, quick addition to keep momentum.
- **Soul.MD:** user.soul + onboarding flow.

### Secondary Features

- **Anywhere:** Making it fun by adding Telegram.
    - This part is also annoying to setup, but a big “wow” moment.
    - Simplicity: we keep it separate from chats.
